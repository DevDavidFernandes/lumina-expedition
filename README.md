# ✦ LUMINA  
### Sua expedição pessoal

> “Toda rotina é uma jornada. Toda tarefa, uma marca deixada no caminho.”

---

## 🌘 Sobre o projeto

**Lumina** é uma aplicação web de produtividade gamificada onde sua vida cotidiana é transformada em uma expedição pessoal.

Aqui, tarefas não são apenas tarefas —  
são **missões**.

E seu progresso não é invisível —  
ele se manifesta em **XP, níveis e crescimento contínuo**.

Inspirado na atmosfera melancólica e artística de expedições e registros de jornada, Lumina busca transformar disciplina em narrativa.

---

## 🜁 Conceito

Você é um **expedicionário**.

Seu objetivo não é apenas organizar o dia,  
mas **registrar sua própria travessia**.

Cada ação molda sua progressão:

- ✔ Missões concluídas → XP
- ✔ Hábitos mantidos → chroma
- ✔ Diário escrito → memória da jornada
- ✔ Progressão → evolução do expedicionário

---

## ⚙️ Funcionalidades

### 🧭 Sistema de autenticação (fake)
- Cadastro de usuário
- Login e logout
- Sessão via LocalStorage

---

### ⚔️ Missões (Tasks)
- Criar missões
- Editar missões
- Concluir missões
- Excluir missões
- Sistema de XP por dificuldade

---

### 📖 Diário de expedição
- Criar registros pessoais
- Editar entradas
- Excluir entradas
- Organização por data

---

### 🔁 Hábitos
- Criar hábitos recorrentes
- Marcar conclusão diária
- Sistema de streak (sequência)

---

### ✦ Progressão
- Sistema de XP
- Níveis de expedicionário
- Evolução baseada em ações
